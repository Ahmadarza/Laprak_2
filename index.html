1. Mengapa Menggunakan HTML, CSS, dan JavaScript dalam Pemrograman Web?

1. HTML (HyperText Markup Language)

HTML merupakan dasar dari setiap halaman web. Bahasa ini digunakan untuk menyusun struktur dan konten halaman, seperti teks, gambar, tabel, dan formulir. Tanpa HTML, halaman web tidak memiliki kerangka untuk menampilkan informasi di browser.

2. CSS (Cascading Style Sheets)

CSS memberikan gaya dan tata letak pada halaman web. Dengan CSS, pengembang dapat mengatur warna, font, margin, tata letak, dan elemen desain lainnya. Pemisahan antara konten (HTML) dan desain (CSS) memudahkan pengelolaan serta memastikan tampilan yang konsisten di seluruh halaman web.

3. JavaScript

JavaScript adalah bahasa pemrograman yang memungkinkan interaktivitas di halaman web. Bahasa ini memungkinkan pengembang membuat fitur seperti validasi formulir, animasi, atau aplikasi interaktif. JavaScript menjadikan halaman lebih hidup dan responsif terhadap input pengguna.

2. Bagaimana HTML, CSS, dan JavaScript dapat Diintegrasikan dalam Pemrograman Web?

1. HTML Sebagai Struktur Dasar

HTML mendefinisikan elemen-elemen pada halaman, seperti teks paragraf, gambar, dan tombol. Misalnya, elemen <p>, <img>, digunakan untuk menampilkan teks.

2. CSS untuk Gaya dan Tata Letak

CSS diintegrasikan ke HTML melalui tiga metode utama:

  • Inline: Langsung ditulis pada atribut elemen HTML, misalnya <p style="color: blue;">.
  • Internal: Ditulis dalam tag <style> di bagian <head> dokumen HTML.
  • Eksternal: Menggunakan file terpisah (.css) yang dihubungkan dengan tag <link>.

3. JavaScript untuk Interaktivitas

JavaScript diintegrasikan ke dalam HTML melalui:

  • Internal Script: Ditulis dalam elemen <script> di bagian <head> atau sebelum penutup </body>.
  • Eksternal Script: Menggunakan file .js yang dihubungkan dengan tag <script src="script.js">.
